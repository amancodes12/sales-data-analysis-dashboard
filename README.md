# 📊 Sales Performance Dashboard

## 📌 Overview
This project is an end-to-end data analysis solution using Excel, SQL, and Power BI.

The goal was to analyze sales data and generate meaningful business insights.

---

## 🛠 Tools Used
- Excel (Data Cleaning & Pivot Tables)
- SQL (Data Querying & Analysis)
- Power BI (Dashboard Visualization)

---

## 📊 Key Metrics
- Total Sales: 3.60M
- Total Orders: 1330
- Total Profit: 297K
- Profit Margin: 8%

---

## 🔍 Key Insights
- Technology category generates highest sales
- West region is top-performing
- Sales show a declining trend over time
- Top products contribute major revenue
